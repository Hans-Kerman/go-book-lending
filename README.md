# go-book-lending
图书借阅管理系统，go后端自学检测项目。后端我用go编写，前端由ai编写

后端框架使用Gin，ORM使用Gorm(数据库使用PostgreSQL)，缓存使用Redis:7.2-alpine，使用JWT鉴权进行综合练习。

基础的模型定义：

**书籍数据**
- 标题
- 作者
- ISBN
- (赔偿)金额    (待定备用)
- 可借数量
- 封面图片路径
  
**用户数据**
- 用户名
- 角色(管理员、读者)
- 密码
  
**借书记录数据**
- 编号
- 创建时间
- 归还时间(*time.Time，未归还时nil)
- 借书人
- 书本ISBN
